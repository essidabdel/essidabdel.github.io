## ğŸš€ Quick Start

### Prerequisites

Ensure you have the following installed:
- [Git](https://git-scm.com/downloads) - Version control system
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/essidabdel/portfolio.git
```

2. **Navigate to the project directory**

```bash
cd portfolio
```

3. **Open in your browser**

Simply open `index.html` in your preferred web browser, or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using PHP
php -S localhost:8000

# Using Node.js (http-server)
npx http-server
```

4. **Access the portfolio**

Open your browser and navigate to `http://localhost:8000`

## ğŸ¨ Customization

### Changing Colors

Edit CSS variables in `assets/css/style.css`:

```css
:root {
  --orange-yellow-crayola: hsl(45, 100%, 72%); /* Primary color */
  --vegas-gold: hsl(45, 54%, 58%);             /* Secondary color */
  /* Modify other color variables as needed */
}
```

### Updating Content

1. **Personal Information**: Edit `index.html` sidebar section
2. **Projects**: Add/modify project items in the Portfolio section
3. **Experience**: Update timeline items in the Resume section
4. **Images**: Replace files in `assets/images/` directory

### Adding New Sections

Follow the existing HTML structure and add corresponding JavaScript functionality in `script.js` if needed.
